# Agencias #

## Insertar Agencia ##
Para poder insertar una nueva **Agencia** tenemos que presionar el botón que dice
**Agencias**, que encontramos en la barra de navegación, ![ Crear una Agencia ](1.png)
Nos mostrará un submenu que contiene dos opciones:

  * Lista Agencias
  * Nueva Agencia

  ![ Crear una Agencia ](Agregar-Agencia.png)

## Lista de Agencias ##
Para poder visualizar las **Agencias** registradas tendremos que
ir a la barra de navegación, presionar donde dice **Agencias** y presionar
**Lista Agencias**. ![ Lista de Agencias ](ver-Agencias.png)

La página se recargará y nos mostrará un contenido similar a este:
![ Lista de Agencias ](Crud-Agencias.png)

## Editar Agencias ##

Una vez desplegada la **Lista de Agencias** podremos realizar modificaciones
como:

  * Ver a detalle cada Agencia: `View`
  * Editar Agencias: `Edit`
  * Eliminar Agencias: `Delete`

![ Edición de Agencias ](Crud-Agencias.png)

## Detalles de las Agencias ##

Estando en la **Lista de Agencias**, ![ Lista de Agencias ](Crud-Agencias.png)
Podremos visualizar los **Detalles de las Agencias** Presionando `View`.
Nos redirigirá a la siguiente página:
![ Detalles de la Agencia ](View-Agencia.png)

## Eliminar Agencia ##

Para eliminar una agencia debemos presionar `Delete` estando en **Lista de Agencias**,
![ Lista de Agencias ](Crud-Agencias.png) Y nos saldrá una alerta de confirmación
para eliminar la Agencia. ![ Eliminar una Agencia ](Eliminar-Agencia.png)
